Guia rapida

Pasos cortos para mantener Dominican Hair Salon actualizada desde el telefono.

Reservas

  1. Busca por nombre, telefono, email o servicio.
  2. Usa Confirmar, Cancelar o Pendiente para cambiar el estado.
  3. Guarda notas internas si necesitas recordar detalles de la clienta.
  4. Abre PDF para ver o compartir el recibo.

Usuarios

  1. Admin y Developer pueden crear usuarios y decidir permisos.
  2. Los usuarios staff solo editan citas asignadas a su usuario.
  3. Las ganancias staff no incluyen propinas ni descuentos.

Dias libres

  1. El staff puede pedir dias libres desde su acceso.
  2. Admin revisa la solicitud y la aprueba o deniega.
  3. Cada solicitud se guarda en la hoja Dias Libres.

Servicios

  1. Agrega categoria, nombre, descripcion, precio y duracion.
  2. Marca Visible para que aparezca en la pagina de clientas.
  3. Usa Orden para controlar como se muestra cada servicio.

Fotos

  1. Selecciona una foto desde la galeria del telefono.
  2. Elige recorte y ajusta la posicion antes de guardar.
  3. Arrastra filas para cambiar el orden.
  4. Borra fotos que ya no quieras mostrar.

Ajustes

  1. Cambia telefono, email, Instagram y horario.
  2. Bloquea dias cerrados o vacaciones.
  3. Usa Actualizar pagina clientas despues de editar fotos o servicios.
  4. Usa Cerrar sesion cuando termines.
